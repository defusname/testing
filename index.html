Основные функции расширения

  • 1) Быстрый доступ к дальним страницам магазина:

    При клике на расширение открывается блок с удобным списком страниц магазина. Основное предназначение расширения — это быстрый доступ установке конкретных платёжек (теперь не нужно ждать загрузки раздела "Все расширения" и искать нужное приложение!), но вы также можете добавлять свои пункты в других категориях.

  • 2) Открытие конкретной страницы панели администратора в JSON/XML для просмотра детальной информации:

    Просто щелкните правой кнопкой мыши на странице, наведите на "inSales Support" и выберите один из пунктов. Вам не нужно будет вручную менять ссылку с admin2 на admin и подставлять нужное расширение - это произойдёт в два щелчка

  • 3) Вывод id вариантов товара:

    На странице товара отображается id вариантов. Клик левой кнопкой мыши по ID варианта - добавляет его в буфер обмена, клик правой кнопкой мыши+"Открыть JSON"\"Открыть XML" - откроет детальную информацию по выбранному варианту товара (функционал из пункта 2)

  • 4) Быстрая вставка ссылки на нужный раздел:

    В тикетке можно вызвать блок с поиском по разделам панели администратора. Щелчок по нужному разделу сразу же добавляет текстовый путь до него и ссылку на раздел в этом магазине

  • 5) Режим скриншота для администратора:

    В панели администратора используйте сочетание клавиш Ctrl+Shift+X, чтобы включить режим скриншота. В этом режиме будут скрыты элементы, предназначенные только для технической поддержки.

  • Как установить?

    Скачать архив с поста, зайти в chrome://extensions/, включить "Режим разработчика" и перекинуть архив в эту вкладку

    Видео - https://skrinshoter.ru/vMtMhdiXErX?a

  • Как работают категории и страницы в самом расширении?

    По умолчанию всегда есть категория "Вне категории" (туда будут попадать все страницы без категории) и "Приложения". При первой установке так же подтянется ещё пара категорий и страниц, до которых слишком далеко идти вручную (по моему мнению) - их можно удалить. То что находится в "Приложения" добавлять/редактировать и удалять не рекомендуется:)

    Нажимать на нужный раздел для перехода нужно открыв магазин, и дождавшись, когда его url сменится на myinsales.ru (при этом дожидаться окончательной загрузке необязательно)

    Всё по классике - левая кнопка мыши изменит url текущей вкладки, средняя кнопка мыши откроет новую вкладку с изменённым url

  • Что обновляется автоматически?

    Список платёжек (из функционала 1)

    Cписок разделов магазина (из функциоанала 4)

    Cписок элементов, которые скрываются в режиме скриншота (из функционала 5)
